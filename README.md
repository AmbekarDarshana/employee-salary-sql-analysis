Employee Salary & Department Insights (SQL)

Overview
This project analyzes employee salary data to extract department-level insights and identify salary distribution patterns using SQL.

Tables Used:
employees

id
name
department

salaries

emp_id
salary

Business Questions Solved:

Calculated average salary per department
Identified highest paid employee
Found employees earning above company average salary
Categorized employees into salary bands using CASE logic

Key Insights:

The IT department had the highest average salary among all departments.
Salary distribution analysis identified top earners within the organization.
Employees earning above the company average were primarily concentrated in one department.
Salary band categorization (Senior / Mid / Junior) provided a clear structure for workforce segmentation.
Analysis highlighted potential salary imbalance across departments.


SQL Concepts Used:

INNER JOIN
GROUP BY
HAVING
Aggregate Functions (AVG, MAX)
Subqueries
CASE Statements
